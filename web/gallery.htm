Advanced Distributed Learning's (ADL) sandbox application allows the user to create and edit their own environment. Be sure to check out some of the demo environments - you can reach them from the "Visit an Existing World" button.


Kimray Valve

Launch New Session

KimRay Valve, built for the New Mexico Junior College, is a live 3D Immersive Maintenance Demonstration built on top of Virtual World Framework to demonstrate the capabilities of Remote Subject Matter Experts interacting with a student using a shared real time lesson workspace. This application utilizes WebRTC to deliver real time video and voice over IP without any browser plugins.


Additional Users

Advanced Distributed Learning


Advanced Distributed Learning (ADL) has utilized VWF to create a sandbox application that allows the user to create a world directly in the browser. Additionally, the ADL server allows for persistent worlds that can be shared with others over time. Check out their Sandbox application above.


Cole Engineering Services, Inc. (CESI)


Cole Engineering Services, Inc. (CESI) has explored the integration of VWF with JBUS and the EDGE platform.


Lively


Lively Kernel, a new approach to web programming, provides a complete platform for web applications, including dynamic graphics, network access, and development tools. Lively and VWF have teamed up to explore integration between the two architectures. Current exploration and further information can be found on GitHub.


Lockheed Martin


Lockheed Martin is internally developing projects with the Virtual World Framework for maintenance and task training in domains such as Defense, Marine, and Energy.


New Mexico Junior College


New Mexico Junior College (NMJC) is taking advantage of the remote collaborate nature of the framework, creating maintenance training applications that support real time interactions with instructors. Check out their Kimray Valve application above.


Z Shift


Z Shift is developing VWF applications for a virtual command center in correlation with the US Joint and Coalition Warfighting (JCW) group.
