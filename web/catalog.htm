The bzflag application is the Virtual World Framework's version of the classic arcade game. Use the WASD keys to navigate the scene and the Enter key to fire a laser. Use the chat feature to communicate and taunt your opponents, but most of all look out for opposing tanks!

The VWF's take on the class Minesweeper game. Click tiles to turn them over. The number that appears is representative of the number of neighboring mines. Unveiling all numbers wins the game - select the wrong tile and it's game over.

Launch New Session

The tile puzzle application allows a user to shuffle the board or solve the board automatically. It also allows the user to play by clicking on a tile and the tile will move to the open space. The goal of the game is to have the numbers show in sequential order from left to right, top to bottom.
